1

Statistics Basics

Task Definition: Learned mean, median, mode, variance, and standard deviation.

Mean

Average value

Median

Middle value

Mode

Most frequent

Variance

Spread measure

Std Dev

√Variance

import numpy as np

data = [10, 20, 30, 40, 50]

# Mean
mean = np.mean(data)  # 30

# Median
median = np.median(data)  # 30

# Mode (using scipy)
from scipy import stats
mode = stats.mode(data)

# Variance
variance = np.var(data)

# Standard Deviation
std_dev = np.std(data)
